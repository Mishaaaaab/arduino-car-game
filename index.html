Arduino Car Game

Educational robotics project

About the Project

Arduino Car Game is a simple Arduino-based game developed for educational purposes. The player controls a car by moving it up or down using buttons in order to avoid obstacles.

Game Rules

  • Press the L or R button to start the game
  • Move the car up and down
  • Avoid obstacles
  • Collision results in Game Over
