ARDUINO-CAR-GAME

A simple Arduino-based car game: move the car up/down to avoid obstacles. Collision ends the game.

Platform: Arduino
Controls: 2 buttons
Repository: GitHub

Overview

This project demonstrates a minimal game loop on Arduino with button input and simple obstacle logic. The goal is to practice embedded programming, I/O handling, and basic game state management.

  • Press L or R to start
  • Move the car up/down to avoid obstacles
  • Collision => Game Over
